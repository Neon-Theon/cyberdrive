**Built entirely in Google AI Studio with Gemini**  
A high-speed cyberpunk arcade racing game with AI-generated racetracks, missions, vehicles and career progression. Customize your ride in the garage, race through neon-drenched cities, and level up your driving skills!

## ✨ How to Run
1. `npm install`
2. Copy `.env.example` to `.env.local` and add your `GEMINI_API_KEY`
3. `npm run dev`
4. Open http://localhost:5173

## 🛠️ Tech Stack
- Google AI Studio + Gemini
- Vite + React + TypeScript
